The Code is structured in three functions.

Get Values

Get Values stores the start and end values entered by the user. It converts each string value into an integer, if the input can be converted. If the input cannot be converted to an integer, an alert is called to enter integers only.

After the inputs are converted to integers, the generateNumbers and displayNumbers functions are both called.

Generate Numbers

Generate Numbers initializes an empty array to store integers. The user inputs are used as intial and end parameters for a for loop.

The loop increments each index value by one. Each index value is pushed to the numbers array. The numbers array is returned.

Display Numbers

Display Numbers initializes an empty string stored in the variable templateRows. A for loop cycles through the numbers array.

Each number is divided by two to determine if it's even or odd. An "even" or "odd" class is then added to the td. Finally, all rows are displayed in the table.
